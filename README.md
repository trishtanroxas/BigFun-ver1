# 🎉 BigFun Web Application (ver1)

## 📌 Description

**BigFun** is a web-based application developed as a **thesis project**, designed to provide an interactive and efficient platform for users. This system demonstrates the implementation of core web development concepts using **pure PHP (no frameworks)**, focusing on functionality, simplicity, and performance.

---

## 🚀 Features

* 🔐 User Authentication System (Login/Session Handling)
* 📩 Email Notifications using PHPMailer
* 📊 Dynamic Content Management
* 🧩 Modular PHP Structure (Custom MVC-inspired setup)
* 🔄 CRUD Operations (Create, Read, Update, Delete)
* 🎯 User-Friendly Interface

---

## 🛠️ Technologies Used

* **Backend:** PHP (Vanilla PHP)
* **Frontend:** HTML, CSS, JavaScript
* **Email Service:** PHPMailer
* **Database:** MySQL
* **Version Control:** Git / GitHub

---

## ⚙️ Installation Guide

1. Clone the repository:

```bash
git clone https://github.com/your-username/BigFun-ver1.git
```

2. Move the project to your local server:

* (XAMPP / Laragon / WAMP)

3. Import the database:

* Open phpMyAdmin
* Create a database
* Import the `.sql` file

4. Configure database connection:

* Edit your config file (e.g., `connect.php`)

5. Run the project:

```
http://localhost/BigFun-ver1/
```

---

## 🔐 Admin Access

> Admin accounts are manually configured in the database.

* No public admin registration is available
* Access is controlled via role-based authentication
